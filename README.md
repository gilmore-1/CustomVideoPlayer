# VideoPlayer Component

A fully responsive and interactive React video player component that supports dynamic quality switching (`360p` and `720p`) while preserving playback state.

---

## Features

- Switch video quality between `360p` and `720p`.
- Retain the current playback position when switching quality.
- Display current playback time and video duration in a user-friendly format.
- Customizable UI with `styled-components`.
